## 🎯 System Overview

### What We're Building
A sophisticated NBA live betting dashboard that combines:
- **Real-time odds feeds** from The Odds API
- **Live game stats** (ESPN GameCast style)
- **Multiple betting models** with intelligent model selection
- **Edge detection** highlighting betting opportunities
- **Multi-game monitoring** - watch many games at once
- **Unit sizing recommendations** based on edge confidence
- **Sportsbook affiliate integration** for monetization

### Core Value Proposition
The tool does all the work - users just watch for highlighted opportunities. The system calculates true odds, compares to market lines, and recommends bet size based on perceived edge.

### Technology Stack
- **Backend**: FastAPI (Python)
- **Frontend**: React + TypeScript + Tailwind CSS
- **Data Sources**: The Odds API (expandable to Sportradar, ESPN)
- **Models**: Pace-based, Momentum, Regression, Situational, ML ensemble

### Tracking Implementation

```typescript
// Sub-ID tracking for detailed analytics
const trackingCodes = {
  gameCard: 'gc',
  opportunityList: 'opp',
  comparisonTable: 'comp',
  educationalArticle: 'edu',
  headerCTA: 'nav',
  emailAlert: 'email'
};

function buildAffiliateUrl(bookmaker: string, source: string, gameId?: string) {
  const baseUrl = getAffiliateUrl(bookmaker);
  const subId = `${source}_${gameId || 'home'}_${Date.now()}`;
  return `${baseUrl}?sub_id=${subId}`;
}
```
