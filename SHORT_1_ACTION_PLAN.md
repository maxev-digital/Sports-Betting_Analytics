- Export vertical (9:16)

### Option B: SCREEN RECORDING (Better - 45 minutes)

**Using OBS Studio (Free):**

1. **Download OBS Studio**
   - Go to obsproject.com
   - Download and install

2. **Set Up Vertical Recording**
   - Open OBS
   - Settings → Video → Canvas: 1080x1920
   - Add Source: "Display Capture"

3. **Record Each Segment:**
   ```
   Segment 1: Show red alert (3 sec)
   Segment 2: Show odds comparison (5 sec)
   Segment 3: Show calculator (7 sec)
   Segment 4: Show your platform (7 sec)
   Segment 5: Show alert + pricing (6 sec)
   Segment 6: Show logo + URL (2 sec)
   ```

4. **Save Each Recording**
   - File → Start Recording
   - Record segment
   - File → Stop Recording
   - Save as `seg1.mp4`, `seg2.mp4`, etc.
