# 🤖 Custom Betting Automation System - Complete Build Guide

## 📋 Overview

Build a **legal, ToS-compliant** semi-automated betting system that:
- Detects high-value opportunities (arbitrage, goalie pulls, +EV bets)
- Pre-fills bet slips across multiple sportsbooks
- Allows one-click confirmation (user still confirms manually)
- Tracks all bets and results automatically

**Key Principle**: We automate everything EXCEPT the final "Place Bet" click (required by law/ToS).

---

```python
"""
Detect betting opportunities and prepare automation payloads
"""

from typing import List, Dict, Optional
from dataclasses import dataclass
import asyncio
from datetime import datetime

@dataclass
class BettingOpportunity:
    """Standardized betting opportunity for automation"""
    opportunity_id: str
    opportunity_type: str  # "arbitrage", "goalie_pull", "positive_ev"
    sport: str
    
    # Event details
    game_id: str
    home_team: str
    away_team: str
    commence_time: datetime
    
    # Betting instructions
    legs: List[Dict]  # Multiple bets for arbs, single bet for +EV
    
    # Value metrics
    total_stake: float
    guaranteed_profit: Optional[float]  # For arbs
    expected_value: Optional[float]  # For +EV bets
    roi_percentage: float
    
    # Urgency
    priority: str  # "HIGH", "MEDIUM", "LOW"
    expiry_time: datetime  # When opportunity expires
    time_to_place: int  # Seconds available to place bet
    
    # Tracking
    created_at: datetime
    status: str  # "active", "placed", "expired", "error"

@dataclass
class BetLeg:
    """Individual bet in an opportunity"""
    sportsbook: str  # "draftkings", "fanduel", etc.
    market_type: str  # "moneyline", "spread", "total", "team_total"
    selection: str  # "Lakers -4.5", "Over 220.5", etc.
    odds: int  # American odds
    stake: float
    
    # Navigation data
    game_url: str  # Direct link to game on sportsbook
    bet_selector: str  # CSS selector for bet button
    stake_input_selector: str  # CSS selector for stake input

class OpportunityDetector:
    """Main engine for detecting betting opportunities"""
    
    def __init__(self, db, odds_api, stats_db):
        self.db = db
        self.odds_api = odds_api
        self.stats_db = stats_db
        
        # Active opportunities
        self.active_opportunities: List[BettingOpportunity] = []
    
    async def scan_for_opportunities(self):
        """
        Main loop - scan for all opportunity types
        Runs every 5 seconds
        """
        while True:
            try:
                opportunities = []
                
                # 1. Scan for arbitrage
                arbs = await self.scan_arbitrage()
                opportunities.extend(arbs)
                
                # 2. Scan for goalie pull opportunities
                goalie_pulls = await self.scan_goalie_pulls()
                opportunities.extend(goalie_pulls)
                
                # 3. Scan for positive EV bets
                ev_bets = await self.scan_positive_ev()
                opportunities.extend(ev_bets)
                
                # 4. Update active opportunities
                self.active_opportunities = opportunities
                
                # 5. Broadcast to connected clients
                await self.broadcast_opportunities(opportunities)
                
                await asyncio.sleep(5)
                
            except Exception as e:
                print(f"Error in opportunity scan: {e}")
                await asyncio.sleep(5)
    
    async def scan_arbitrage(self) -> List[BettingOpportunity]:
        """Scan for arbitrage opportunities"""
        opportunities = []
        
        # Get all live and upcoming games
        games = await self.odds_api.get_all_games()
        
        for game in games:
            # Get odds from all sportsbooks
            all_odds = await self.odds_api.get_game_odds(game['id'])
            
            # Check for 2-way arbitrage (moneyline)
            arb = self.check_two_way_arbitrage(game, all_odds, 'h2h')
            if arb:
                opportunities.append(arb)
            
            # Check for total arbitrage
            arb = self.check_two_way_arbitrage(game, all_odds, 'totals')
            if arb:
                opportunities.append(arb)
            
            # Check for spread arbitrage
            arb = self.check_two_way_arbitrage(game, all_odds, 'spreads')
            if arb:
                opportunities.append(arb)
        
        return opportunities
    
    def check_two_way_arbitrage(
        self, 
        game: Dict, 
        all_odds: List[Dict],
        market_type: str
    ) -> Optional[BettingOpportunity]:
        """
        Check for arbitrage in 2-way markets
        """
        if market_type == 'h2h':
            # Moneyline
            best_home = self.find_best_odds(all_odds, 'home', market_type)
            best_away = self.find_best_odds(all_odds, 'away', market_type)
            
            if not (best_home and best_away):
                return None
            
            # Calculate if arbitrage exists
            home_decimal = self.american_to_decimal(best_home['odds'])
            away_decimal = self.american_to_decimal(best_away['odds'])
            
            total_implied = (1 / home_decimal) + (1 / away_decimal)
            
            if total_implied >= 1.0:
                return None  # No arbitrage
            
            # Calculate stakes for $1,000 guaranteed return
            guaranteed_return = 1000
            home_stake = guaranteed_return / home_decimal
            away_stake = guaranteed_return / away_decimal
            total_stake = home_stake + away_stake
            profit = guaranteed_return - total_stake
            roi = (profit / total_stake) * 100
            
            # Only alert if ROI >= 1%
            if roi < 1.0:
                return None
            
            # Create opportunity
            return BettingOpportunity(
                opportunity_id=f"arb_{game['id']}_{market_type}_{int(datetime.now().timestamp())}",
                opportunity_type="arbitrage",
                sport=game['sport_key'],
                game_id=game['id'],
                home_team=game['home_team'],
                away_team=game['away_team'],
                commence_time=datetime.fromisoformat(game['commence_time'].replace('Z', '+00:00')),
                legs=[
                    {
                        'sportsbook': best_home['bookmaker'],
                        'market_type': 'moneyline',
                        'selection': game['home_team'],
                        'odds': best_home['odds'],
                        'stake': round(home_stake, 2),
                        'game_url': self.generate_game_url(best_home['bookmaker'], game),
                        'bet_selector': self.get_bet_selector(best_home['bookmaker'], 'moneyline', 'home'),
                        'stake_input_selector': self.get_stake_selector(best_home['bookmaker'])
                    },
                    {
                        'sportsbook': best_away['bookmaker'],
                        'market_type': 'moneyline',
                        'selection': game['away_team'],
                        'odds': best_away['odds'],
                        'stake': round(away_stake, 2),
                        'game_url': self.generate_game_url(best_away['bookmaker'], game),
                        'bet_selector': self.get_bet_selector(best_away['bookmaker'], 'moneyline', 'away'),
                        'stake_input_selector': self.get_stake_selector(best_away['bookmaker'])
                    }
                ],
                total_stake=round(total_stake, 2),
                guaranteed_profit=round(profit, 2),
                expected_value=None,
                roi_percentage=round(roi, 2),
                priority="HIGH" if roi >= 3.0 else "MEDIUM",
                expiry_time=datetime.now() + timedelta(minutes=5),
                time_to_place=300,
                created_at=datetime.now(),
                status="active"
            )
        
        # Similar logic for totals and spreads...
        return None
    
    async def scan_goalie_pulls(self) -> List[BettingOpportunity]:
        """Scan for NHL goalie pull opportunities"""
        opportunities = []
        
        # Get live NHL games
        nhl_games = await self.odds_api.get_live_games(sport='ice_hockey')
        
        for game in nhl_games:
            # Only check 3rd period
            if game.get('period', 0) < 3:
                continue
            
            # Check if team is trailing and likely to pull goalie
            prediction = await self.predict_goalie_pull(game)
            
            if not prediction:
                continue
            
            # Check if pull is imminent (30-90 seconds)
            if not (30 <= prediction['time_until_pull'] <= 90):
                continue
            
            # Get best odds for OVER
            odds = await self.odds_api.get_game_odds(game['id'])
            best_over = self.find_best_over_odds(odds)
            
            if not best_over or best_over['odds'] < 110:
                continue  # Need plus money
            
            # Calculate EV
            ev_analysis = await self.calculate_goalie_pull_ev(game, best_over, prediction)
            
            if ev_analysis['expected_value'] < 5.0:
                continue  # Need 5%+ EV
            
            # Create opportunity
            opportunities.append(BettingOpportunity(
                opportunity_id=f"goalie_pull_{game['id']}_{int(datetime.now().timestamp())}",
                opportunity_type="goalie_pull",
                sport="ice_hockey",
                game_id=game['id'],
                home_team=game['home_team'],
                away_team=game['away_team'],
                commence_time=datetime.fromisoformat(game['commence_time'].replace('Z', '+00:00')),
                legs=[{
                    'sportsbook': best_over['bookmaker'],
                    'market_type': 'total',
                    'selection': f"Over {best_over['point']}",
                    'odds': best_over['odds'],
                    'stake': 100.0,  # User can adjust
                    'game_url': self.generate_game_url(best_over['bookmaker'], game),
                    'bet_selector': self.get_bet_selector(best_over['bookmaker'], 'total', 'over'),
                    'stake_input_selector': self.get_stake_selector(best_over['bookmaker'])
                }],
                total_stake=100.0,
                guaranteed_profit=None,
                expected_value=ev_analysis['expected_value'],
                roi_percentage=ev_analysis['expected_value'],
                priority="HIGH",
                expiry_time=datetime.now() + timedelta(seconds=prediction['time_until_pull']),
                time_to_place=prediction['time_until_pull'],
                created_at=datetime.now(),
                status="active"
            ))
        
        return opportunities
    
    def american_to_decimal(self, odds: int) -> float:
        """Convert American odds to decimal"""
        if odds > 0:
            return (odds / 100) + 1
        else:
            return (100 / abs(odds)) + 1
    
    def generate_game_url(self, bookmaker: str, game: Dict) -> str:
        """Generate direct URL to game on sportsbook"""
        
        # URL templates for each sportsbook
        templates = {
            'draftkings': f"https://sportsbook.draftkings.com/event/{game['id']}",
            'fanduel': f"https://sportsbook.fanduel.com/event/{game['id']}",
            'betmgm': f"https://sports.betmgm.com/event/{game['id']}",
            # Add more...
        }
        
        return templates.get(bookmaker, "#")
    
    def get_bet_selector(self, bookmaker: str, market: str, selection: str) -> str:
        """Get CSS selector for bet button"""
        
        # This would need to be maintained as sportsbooks change their HTML
        selectors = {
            'draftkings': {
                'moneyline': {
                    'home': '.event-cell.home .sportsbook-outcome-cell__label',
                    'away': '.event-cell.away .sportsbook-outcome-cell__label'
                },
                'total': {
                    'over': '.sportsbook-outcome-cell[aria-label*="Over"]',
                    'under': '.sportsbook-outcome-cell[aria-label*="Under"]'
                }
            },
            # Add more sportsbooks...
        }
        
        return selectors.get(bookmaker, {}).get(market, {}).get(selection, '')
    
    def get_stake_selector(self, bookmaker: str) -> str:
        """Get CSS selector for stake input field"""
        
        selectors = {
            'draftkings': 'input[placeholder="Enter Wager"]',
            'fanduel': 'input[data-test-id="Betslip-StakeInput"]',
            'betmgm': 'input[name="stake"]',
            # Add more...
        }
        
        return selectors.get(bookmaker, '')
```

```python
"""
WebSocket server for real-time opportunity broadcasting
"""

from fastapi import WebSocket, WebSocketDisconnect
from typing import List, Set
import json

class ConnectionManager:
    """Manage WebSocket connections"""
    
    def __init__(self):
        self.active_connections: Set[WebSocket] = set()
    
    async def connect(self, websocket: WebSocket):
        await websocket.accept()
        self.active_connections.add(websocket)
        print(f"Client connected. Total: {len(self.active_connections)}")
    
    def disconnect(self, websocket: WebSocket):
        self.active_connections.remove(websocket)
        print(f"Client disconnected. Total: {len(self.active_connections)}")
    
    async def broadcast(self, message: dict):
        """Broadcast message to all connected clients"""
        disconnected = set()
        
        for connection in self.active_connections:
            try:
                await connection.send_json(message)
            except Exception as e:
                print(f"Error sending to client: {e}")
                disconnected.add(connection)
        
        # Remove disconnected clients
        self.active_connections -= disconnected

# Global manager instance
manager = ConnectionManager()

# Add to FastAPI app
from fastapi import FastAPI

app = FastAPI()

@app.websocket("/ws/opportunities")
async def websocket_endpoint(websocket: WebSocket):
    await manager.connect(websocket)
    try:
        while True:
            # Keep connection alive
            data = await websocket.receive_text()
            
            # Client can send requests
            if data == "get_opportunities":
                # Send current opportunities
                await websocket.send_json({
                    'type': 'opportunities',
                    'data': [opp.__dict__ for opp in detector.active_opportunities]
                })
    
    except WebSocketDisconnect:
        manager.disconnect(websocket)

# Broadcast function for opportunity detector
async def broadcast_opportunities(opportunities: List[BettingOpportunity]):
    """Called by opportunity detector when new opportunities found"""
    
    message = {
        'type': 'new_opportunities',
        'timestamp': datetime.now().isoformat(),
        'count': len(opportunities),
        'opportunities': [
            {
                'id': opp.opportunity_id,
                'type': opp.opportunity_type,
                'sport': opp.sport,
                'game': f"{opp.away_team} @ {opp.home_team}",
                'roi': opp.roi_percentage,
                'priority': opp.priority,
                'time_to_place': opp.time_to_place,
                'legs': opp.legs
            }
            for opp in opportunities
            if opp.status == 'active'
        ]
    }
    
    await manager.broadcast(message)
```

### Background Script: `extension/background.js`

```javascript
/**
 * Background service worker - manages WebSocket connection
 */

let socket = null;
let reconnectInterval = null;

// Connect to backend WebSocket
function connectWebSocket() {
  socket = new WebSocket('ws://localhost:8000/ws/opportunities');
  
  socket.onopen = () => {
    console.log('Connected to opportunity server');
    clearInterval(reconnectInterval);
    
    // Show notification
    chrome.notifications.create({
      type: 'basic',
      iconUrl: 'icons/icon48.png',
      title: 'Betting Assistant Active',
      message: 'Connected to opportunity scanner'
    });
  };
  
  socket.onmessage = (event) => {
    const message = JSON.parse(event.data);
    
    if (message.type === 'new_opportunities') {
      handleNewOpportunities(message.opportunities);
    }
  };
  
  socket.onerror = (error) => {
    console.error('WebSocket error:', error);
  };
  
  socket.onclose = () => {
    console.log('Disconnected from server. Reconnecting...');
    reconnectInterval = setInterval(connectWebSocket, 5000);
  };
}

// Handle new opportunities
function handleNewOpportunities(opportunities) {
  opportunities.forEach(opp => {
    // Show notification
    chrome.notifications.create({
      type: 'basic',
      iconUrl: 'icons/icon48.png',
      title: `${opp.type.toUpperCase()} - ${opp.priority}`,
      message: `${opp.game}\nROI: ${opp.roi}%\nTime: ${opp.time_to_place}s`,
      buttons: [
        { title: 'Place Bets' },
        { title: 'Dismiss' }
      ]
    }, (notificationId) => {
      // Store opportunity data
      chrome.storage.local.set({
        [notificationId]: opp
      });
    });
  });
  
  // Store in extension storage
  chrome.storage.local.set({
    opportunities: opportunities
  });
}

// Handle notification clicks
chrome.notifications.onButtonClicked.addListener((notificationId, buttonIndex) => {
  if (buttonIndex === 0) {
    // "Place Bets" clicked
    chrome.storage.local.get([notificationId], (result) => {
      const opportunity = result[notificationId];
      if (opportunity) {
        placeBetsSequence(opportunity);
      }
    });
  }
  
  // Clear notification
  chrome.notifications.clear(notificationId);
});

// Place bets in sequence
async function placeBetsSequence(opportunity) {
  for (const leg of opportunity.legs) {
    // Open sportsbook tab
    const tab = await chrome.tabs.create({
      url: leg.game_url,
      active: true
    });
    
    // Wait for page to load
    await new Promise(resolve => {
      chrome.tabs.onUpdated.addListener(function listener(tabId, info) {
        if (tabId === tab.id && info.status === 'complete') {
          chrome.tabs.onUpdated.removeListener(listener);
          resolve();
        }
      });
    });
    
    // Send bet data to content script
    await chrome.tabs.sendMessage(tab.id, {
      action: 'fill_bet_slip',
      leg: leg
    });
    
    // Wait for user to confirm bet
    await new Promise(resolve => setTimeout(resolve, 10000)); // 10 second delay
  }
}

// Initialize on install
chrome.runtime.onInstalled.addListener(() => {
  connectWebSocket();
});

// Initialize on startup
connectWebSocket();
```

```javascript
/**

// Listen for messages from background script
chrome.runtime.onMessage.addListener((request, sender, sendResponse) => {
  if (request.action === 'fill_bet_slip') {
    fillBetSlip(request.leg);
    sendResponse({ status: 'success' });
  }
});

// Auto-fill bet slip
async function fillBetSlip(leg) {
  console.log('Filling bet slip:', leg);
  
  // Wait for page to be fully loaded
  await waitForElement('body');
  
  // Click on the bet button
  const betButton = document.querySelector(leg.bet_selector);
  if (betButton) {
    betButton.click();
    console.log('Clicked bet button');
    
    // Wait for bet slip to appear
    await sleep(1000);
    
    // Fill in stake amount
    const stakeInput = document.querySelector(leg.stake_input_selector);
    if (stakeInput) {
      // Clear existing value
      stakeInput.value = '';
      stakeInput.focus();
      
      // Type stake amount
      stakeInput.value = leg.stake.toString();
      
      // Trigger input event
      stakeInput.dispatchEvent(new Event('input', { bubbles: true }));
      stakeInput.dispatchEvent(new Event('change', { bubbles: true }));
      
      console.log(`Filled stake: $${leg.stake}`);
      
      // Highlight the "Place Bet" button
      await sleep(500);
      highlightPlaceBetButton();
    } else {
      console.error('Could not find stake input');
    }
  } else {
    console.error('Could not find bet button');
  }
}

// Highlight the place bet button
function highlightPlaceBetButton() {
  // Find place bet button (varies by sportsbook)
  const selectors = [
    'button[data-test-id="Place Bet"]',
    'button[class*="place-bet"]',
    'button[class*="submit-bet"]',
    'button:contains("Place Bet")'
  ];
  
  for (const selector of selectors) {
    const button = document.querySelector(selector);
    if (button) {
      // Add pulsing highlight
      button.style.animation = 'pulse 1s infinite';
      button.style.boxShadow = '0 0 20px 5px #00ff00';
      
      // Add CSS animation
      if (!document.getElementById('betting-assistant-styles')) {
        const style = document.createElement('style');
        style.id = 'betting-assistant-styles';
        style.textContent = `
          @keyframes pulse {
            0%, 100% { transform: scale(1); }
            50% { transform: scale(1.05); }
          }
        `;
        document.head.appendChild(style);
      }
      
      console.log('Place bet button highlighted');
      break;
    }
  }
}

// Helper: wait for element to appear
function waitForElement(selector, timeout = 10000) {
  return new Promise((resolve, reject) => {
    const element = document.querySelector(selector);
    if (element) {
      resolve(element);
      return;
    }
    
    const observer = new MutationObserver((mutations) => {
      const element = document.querySelector(selector);
      if (element) {
        observer.disconnect();
        resolve(element);
      }
    });
    
    observer.observe(document.body, {
      childList: true,
      subtree: true
    });
    
    setTimeout(() => {
      observer.disconnect();
      reject(new Error(`Element ${selector} not found within ${timeout}ms`));
    }, timeout);
  });
}

// Helper: sleep
function sleep(ms) {
  return new Promise(resolve => setTimeout(resolve, ms));
}
```

### `backend/automation/bet_calculator.py`

```python
"""
Bet calculation utilities
"""

from typing import Tuple, Optional

class BetCalculator:
    """Calculate optimal bet sizes and expected values"""
    
    @staticmethod
    def american_to_decimal(odds: int) -> float:
        """Convert American odds to decimal"""
        if odds > 0:
            return (odds / 100) + 1
        else:
            return (100 / abs(odds)) + 1
    
    @staticmethod
    def american_to_implied_prob(odds: int) -> float:
        """Convert American odds to implied probability"""
        if odds < 0:
            return abs(odds) / (abs(odds) + 100)
        else:
            return 100 / (odds + 100)
    
    @staticmethod
    def calculate_arbitrage_stakes(
        odds_a: int,
        odds_b: int,
        total_stake: float = 1000.0
    ) -> Tuple[float, float, float, float]:
        """
        Calculate stakes for 2-way arbitrage
        
        Returns:
            (stake_a, stake_b, guaranteed_return, profit)
        """
        decimal_a = BetCalculator.american_to_decimal(odds_a)
        decimal_b = BetCalculator.american_to_decimal(odds_b)
        
        # Check if arbitrage exists
        total_implied = (1 / decimal_a) + (1 / decimal_b)
        if total_implied >= 1.0:
            return (0, 0, 0, 0)  # No arbitrage
        
        # Calculate guaranteed return
        guaranteed_return = total_stake
        
        # Calculate stakes
        stake_a = guaranteed_return / decimal_a
        stake_b = guaranteed_return / decimal_b
        
        actual_total_stake = stake_a + stake_b
        profit = guaranteed_return - actual_total_stake
        
        # Scale to requested total stake
        scale_factor = total_stake / actual_total_stake
        stake_a *= scale_factor
        stake_b *= scale_factor
        guaranteed_return *= scale_factor
        profit *= scale_factor
        
        return (
            round(stake_a, 2),
            round(stake_b, 2),
            round(guaranteed_return, 2),
            round(profit, 2)
        )
    
    @staticmethod
    def calculate_ev(
        true_probability: float,
        american_odds: int,
        stake: float = 100.0
    ) -> Tuple[float, float]:
        """
        Calculate expected value
        
        Returns:
            (ev_dollars, ev_percentage)
        """
        decimal_odds = BetCalculator.american_to_decimal(american_odds)
        implied_prob = BetCalculator.american_to_implied_prob(american_odds)
        
        # Calculate potential profit
        potential_profit = stake * (decimal_odds - 1)
        
        # Calculate EV
        ev = (true_probability * potential_profit) - ((1 - true_probability) * stake)
        ev_percentage = (ev / stake) * 100
        
        return (round(ev, 2), round(ev_percentage, 2))
    
    @staticmethod
    def calculate_kelly_stake(
        edge: float,
        odds: int,
        bankroll: float,
        kelly_fraction: float = 0.25
    ) -> float:
        """
        Calculate optimal stake using Kelly Criterion
        
        Args:
            edge: True probability - implied probability
            odds: American odds
            bankroll: Total bankroll
            kelly_fraction: Fraction of Kelly to use (0.25 = quarter Kelly)
        
        Returns:
            Recommended stake amount
        """
        if edge <= 0:
            return 0
        
        decimal_odds = BetCalculator.american_to_decimal(odds)
        
        # Kelly formula
        full_kelly = (edge * decimal_odds) / (decimal_odds - 1)
        
        # Apply fraction
        fractional_kelly = full_kelly * kelly_fraction
        
        # Calculate stake
        stake = bankroll * fractional_kelly
        
        # Cap at 5% of bankroll
        max_stake = bankroll * 0.05
        stake = min(stake, max_stake)
        
        return round(stake, 2)
```

```python
"""
The Odds API integration
"""

import aiohttp
from typing import List, Dict, Optional
from config.settings import settings

class OddsAPI:
    """Interface to The Odds API"""
    
    def __init__(self):
        self.api_key = settings.odds_api_key
        self.base_url = settings.odds_api_base_url
    
    async def get_sports(self) -> List[Dict]:
        """Get list of available sports"""
        url = f"{self.base_url}/sports"
        params = {'apiKey': self.api_key}
        
        async with aiohttp.ClientSession() as session:
            async with session.get(url, params=params) as response:
                return await response.json()
    
    async def get_odds(
        self,
        sport: str,
        regions: str = 'us',
        markets: str = 'h2h,spreads,totals'
    ) -> List[Dict]:
        """
        Get odds for a sport
        
        Args:
            sport: Sport key (e.g., 'basketball_nba')
            regions: Comma-separated regions (e.g., 'us,uk')
            markets: Comma-separated markets (e.g., 'h2h,spreads,totals')
        """
        url = f"{self.base_url}/sports/{sport}/odds"
        params = {
            'apiKey': self.api_key,
            'regions': regions,
            'markets': markets,
            'oddsFormat': 'american'
        }
        
        async with aiohttp.ClientSession() as session:
            async with session.get(url, params=params) as response:
                return await response.json()
    
    async def get_game_odds(self, sport: str, game_id: str) -> Optional[Dict]:
        """Get odds for a specific game"""
        odds = await self.get_odds(sport)
        
        for game in odds:
            if game['id'] == game_id:
                return game
        
        return None
    
    async def get_live_games(self, sport: str) -> List[Dict]:
        """Get currently live games"""
        all_games = await self.get_odds(sport)
        
        # Filter for live games
        live_games = [
            game for game in all_games
            if not game.get('commence_time')  # Live games don't have commence_time
        ]
        
        return live_games
```

```python
"""
Main opportunity detection engine
"""

import asyncio
from typing import List, Optional
from datetime import datetime, timedelta
import uuid

from models.opportunity import BettingOpportunity, BetLeg, OpportunityType, Priority
from automation.bet_calculator import BetCalculator
from api.odds_api import OddsAPI
from api.sportsbook_selectors import get_selector, get_stake_selector, generate_game_url
from config.settings import settings

class OpportunityDetector:
    """Detect betting opportunities across all markets"""
    
    def __init__(self):
        self.odds_api = OddsAPI()
        self.calculator = BetCalculator()
        self.active_opportunities: List[BettingOpportunity] = []
    
    async def scan_loop(self):
        """Main scanning loop"""
        while True:
            try:
                opportunities = []
                
                # Scan NBA
                nba_opps = await self.scan_sport('basketball_nba')
                opportunities.extend(nba_opps)
                
                # Scan NFL
                nfl_opps = await self.scan_sport('americanfootball_nfl')
                opportunities.extend(nfl_opps)
                
                # Scan NHL
                nhl_opps = await self.scan_sport('icehockey_nhl')
                opportunities.extend(nhl_opps)
                
                # Update active opportunities
                self.active_opportunities = opportunities
                
                print(f"Found {len(opportunities)} opportunities")
                
                await asyncio.sleep(5)  # Scan every 5 seconds
                
            except Exception as e:
                print(f"Error in scan loop: {e}")
                await asyncio.sleep(5)
    
    async def scan_sport(self, sport: str) -> List[BettingOpportunity]:
        """Scan a specific sport for opportunities"""
        opportunities = []
        
        # Get odds
        games = await self.odds_api.get_odds(sport)
        
        for game in games:
            # Check for arbitrage
            arb = await self.check_arbitrage(game, sport)
            if arb:
                opportunities.append(arb)
        
        return opportunities
    
    async def check_arbitrage(
        self,
        game: Dict,
        sport: str
    ) -> Optional[BettingOpportunity]:
        """Check for arbitrage opportunities in a game"""
        
        bookmakers = game.get('bookmakers', [])
        if len(bookmakers) < 2:
            return None
        
        # Check moneyline arbitrage
        best_home = self._find_best_odds(bookmakers, 'h2h', 0)
        best_away = self._find_best_odds(bookmakers, 'h2h', 1)
        
        if not (best_home and best_away):
            return None
        
        # Calculate if arbitrage exists
        home_decimal = self.calculator.american_to_decimal(best_home['price'])
        away_decimal = self.calculator.american_to_decimal(best_away['price'])
        
        total_implied = (1 / home_decimal) + (1 / away_decimal)
        
        if total_implied >= 1.0:
            return None  # No arbitrage
        
        # Calculate stakes
        stake_home, stake_away, guaranteed_return, profit = \
            self.calculator.calculate_arbitrage_stakes(
                best_home['price'],
                best_away['price'],
                total_stake=1000.0
            )
        
        roi = (profit / (stake_home + stake_away)) * 100
        
        # Only alert if ROI >= minimum threshold
        if roi < settings.min_arbitrage_roi:
            return None
        
        # Create opportunity
        opportunity_id = f"arb_{uuid.uuid4().hex[:8]}"
        
        legs = [
            BetLeg(
                sportsbook=best_home['bookmaker'],
                market_type='moneyline',
                selection=game['home_team'],
                odds=best_home['price'],
                stake=stake_home,
                game_url=generate_game_url(best_home['bookmaker'], game['id'], sport),
                bet_selector=get_selector(best_home['bookmaker'], 'moneyline', 'home'),
                stake_input_selector=get_stake_selector(best_home['bookmaker']),
                expected_return=guaranteed_return
            ),
            BetLeg(
                sportsbook=best_away['bookmaker'],
                market_type='moneyline',
                selection=game['away_team'],
                odds=best_away['price'],
                stake=stake_away,
                game_url=generate_game_url(best_away['bookmaker'], game['id'], sport),
                bet_selector=get_selector(best_away['bookmaker'], 'moneyline', 'away'),
                stake_input_selector=get_stake_selector(best_away['bookmaker']),
                expected_return=guaranteed_return
            )
        ]
        
        return BettingOpportunity(
            opportunity_id=opportunity_id,
            opportunity_type=OpportunityType.ARBITRAGE,
            sport=sport,
            game_id=game['id'],
            home_team=game['home_team'],
            away_team=game['away_team'],
            commence_time=datetime.fromisoformat(game['commence_time'].replace('Z', '+00:00')),
            legs=legs,
            total_stake=stake_home + stake_away,
            guaranteed_profit=profit,
            roi_percentage=roi,
            priority=Priority.HIGH if roi >= 3.0 else Priority.MEDIUM,
            expiry_time=datetime.now() + timedelta(seconds=settings.opportunity_expiry_seconds),
            time_to_place=300
        )
    
    def _find_best_odds(
        self,
        bookmakers: List[Dict],
        market_key: str,
        outcome_index: int
    ) -> Optional[Dict]:
        """Find best odds for an outcome across bookmakers"""
        
        best_odds = None
        best_price = float('-inf')
        
        for bookmaker in bookmakers:
            markets = bookmaker.get('markets', [])
            
            for market in markets:
                if market['key'] != market_key:
                    continue
                
                outcomes = market.get('outcomes', [])
                if outcome_index >= len(outcomes):
                    continue
                
                outcome = outcomes[outcome_index]
                price = outcome['price']
                
                if price > best_price:
                    best_price = price
                    best_odds = {
                        'bookmaker': bookmaker['key'],
                        'price': price,
                        'name': outcome['name']
                    }
        
        return best_odds
```

```python
"""
WebSocket server for real-time opportunity broadcasting
"""

from fastapi import WebSocket, WebSocketDisconnect
from typing import Set
import json
from datetime import datetime

class ConnectionManager:
    """Manage WebSocket connections"""
    
    def __init__(self):
        self.active_connections: Set[WebSocket] = set()
    
    async def connect(self, websocket: WebSocket):
        await websocket.accept()
        self.active_connections.add(websocket)
        print(f"Client connected. Total: {len(self.active_connections)}")
    
    def disconnect(self, websocket: WebSocket):
        self.active_connections.discard(websocket)
        print(f"Client disconnected. Total: {len(self.active_connections)}")
    
    async def broadcast(self, message: dict):
        """Broadcast message to all connected clients"""
        disconnected = set()
        
        for connection in self.active_connections:
            try:
                await connection.send_json(message)
            except Exception as e:
                print(f"Error sending to client: {e}")
                disconnected.add(connection)
        
        # Remove disconnected clients
        self.active_connections -= disconnected
    
    async def send_opportunities(self, opportunities: list):
        """Send opportunities to all connected clients"""
        message = {
            'type': 'opportunities',
            'timestamp': datetime.now().isoformat(),
            'count': len(opportunities),
            'data': opportunities
        }
        await self.broadcast(message)

# Global manager instance
manager = ConnectionManager()
```

```javascript
/**
 * Background service worker
 */

let socket = null;
let reconnectInterval = null;
const WS_URL = 'ws://localhost:8000/ws/opportunities';

// Connect to backend
function connectWebSocket() {
  socket = new WebSocket(WS_URL);
  
  socket.onopen = () => {
    console.log('Connected to opportunity server');
    clearInterval(reconnectInterval);
    
    chrome.notifications.create({
      type: 'basic',
      iconUrl: 'icons/icon48.png',
      title: 'Betting Assistant Active',
      message: 'Connected and scanning for opportunities'
    });
    
    // Store connection status
    chrome.storage.local.set({ connected: true });
  };
  
  socket.onmessage = (event) => {
    const message = JSON.parse(event.data);
    
    if (message.type === 'opportunities') {
      handleOpportunities(message.data);
    }
  };
  
  socket.onerror = (error) => {
    console.error('WebSocket error:', error);
    chrome.storage.local.set({ connected: false });
  };
  
  socket.onclose = () => {
    console.log('Disconnected. Reconnecting...');
    chrome.storage.local.set({ connected: false });
    reconnectInterval = setInterval(connectWebSocket, 5000);
  };
}

// Handle new opportunities
function handleOpportunities(opportunities) {
  // Store in local storage
  chrome.storage.local.set({ opportunities });
  
  // Send notifications for high priority
  opportunities.forEach(opp => {
    if (opp.priority === 'HIGH' && opp.status === 'active') {
      chrome.notifications.create({
        type: 'basic',
        iconUrl: 'icons/icon48.png',
        title: `${opp.opportunity_type.toUpperCase()} - ${opp.roi_percentage}%`,
        message: `${opp.away_team} @ ${opp.home_team}\n${opp.legs.length} bets • ${opp.time_to_place}s`,
        buttons: [
          { title: 'Place Bets' },
          { title: 'Dismiss' }
        ]
      }, (notificationId) => {
        chrome.storage.local.set({ [notificationId]: opp });
      });
    }
  });
}

// Handle notification clicks
chrome.notifications.onButtonClicked.addListener(async (notificationId, buttonIndex) => {
  if (buttonIndex === 0) {
    const result = await chrome.storage.local.get([notificationId]);
    const opportunity = result[notificationId];
    
    if (opportunity) {
      await placeBetsSequence(opportunity);
    }
  }
  
  chrome.notifications.clear(notificationId);
});

// Place bets in sequence
async function placeBetsSequence(opportunity) {
  for (const leg of opportunity.legs) {
    try {
      // Open tab
      const tab = await chrome.tabs.create({
        url: leg.game_url,
        active: true
      });
      
      // Wait for page load
      await waitForTabLoad(tab.id);
      
      // Send message to content script
      await chrome.tabs.sendMessage(tab.id, {
        action: 'fill_bet_slip',
        leg: leg
      });
      
      // Wait before next bet
      await sleep(8000);
      
    } catch (error) {
      console.error('Error placing bet:', error);
    }
  }
}

// Helper functions
function waitForTabLoad(tabId) {
  return new Promise((resolve) => {
    chrome.tabs.onUpdated.addListener(function listener(id, info) {
      if (id === tabId && info.status === 'complete') {
        chrome.tabs.onUpdated.removeListener(listener);
        resolve();
      }
    });
  });
}

function sleep(ms) {
  return new Promise(resolve => setTimeout(resolve, ms));
}
```

### `extension/popup.js`

```javascript
/**
 * Popup UI logic
 */

document.addEventListener('DOMContentLoaded', () => {
  loadOpportunities();
  setInterval(loadOpportunities, 3000);
});

async function loadOpportunities() {
  const result = await chrome.storage.local.get(['opportunities', 'connected']);
  const opportunities = result.opportunities || [];
  const connected = result.connected || false;
  
  // Update status
  const statusEl = document.getElementById('status');
  statusEl.className = connected ? 
    'status-indicator status-connected' : 
    'status-indicator status-disconnected';
  
  // Update stats
  document.getElementById('opp-count').textContent = opportunities.length;
  
  if (opportunities.length > 0) {
    const avgRoi = opportunities.reduce((sum, opp) => sum + opp.roi_percentage, 0) / opportunities.length;
    document.getElementById('avg-roi').textContent = `${avgRoi.toFixed(1)}%`;
  }
  
  // Render opportunities
  const container = document.getElementById('opportunities-container');
  
  if (opportunities.length === 0) {
    container.innerHTML = `
      <div class="no-opportunities">
        ${connected ? 'No opportunities found' : 'Connecting to server...'}
      </div>
    `;
    return;
  }
  
  container.innerHTML = opportunities.map(opp => `
    <div class="opportunity-card">
      <div class="opportunity-header">
        <span class="opportunity-type priority-${opp.priority}">
          ${opp.opportunity_type}
        </span>
        <span class="roi">${opp.roi_percentage.toFixed(1)}%</span>
      </div>
      
      <div class="game-title">
        ${opp.away_team} @ ${opp.home_team}
      </div>
      
      <div class="game-details">
        ${opp.legs.length} bet${opp.legs.length > 1 ? 's' : ''} • 
        ${opp.total_stake.toFixed(2)} stake •
        ${opp.time_to_place}s remaining
      </div>
      
      <button class="bet-button" data-opp-id="${opp.opportunity_id}">
        Place Bets - Win ${(opp.guaranteed_profit || opp.expected_value || 0).toFixed(2)}
      </button>
    </div>
  `).join('');
  
  // Add click handlers
  document.querySelectorAll('.bet-button').forEach(button => {
    button.addEventListener('click', () => {
      const oppId = button.dataset.oppId;
      const opp = opportunities.find(o => o.opportunity_id === oppId);
      if (opp) {
        chrome.runtime.sendMessage({
          action: 'place_bets',
          opportunity: opp
        });
        window.close();
      }
    });
  });
}
```

# Sports Betting Automation System

A complete betting automation system that detects arbitrage opportunities and automates bet placement across multiple sportsbooks.

## Features

- 🔍 Real-time opportunity detection (arbitrage, +EV, goalie pulls)
- 🤖 Semi-automated bet placement (95% automated, user confirms)
- 📊 WebSocket-based real-time alerts
- 🎯 Support for 5+ major sportsbooks
- 📱 Browser extension for Chrome/Firefox
- 🔒 100% ToS compliant (requires manual confirmation)

## Installation

### Backend Setup

1. Install dependencies:
```bash
cd backend
pip install -r requirements.txt
```

2. Configure environment:
```bash
cp .env.example .env
# Edit .env with your API keys
```

3. Set up database:
```bash
psql -U postgres -f database/schema.sql
```

4. Run server:
```bash
python main.py
```

### Extension Setup

1. Open Chrome
2. Go to `chrome://extensions`
3. Enable "Developer mode"
4. Click "Load unpacked"
5. Select the `extension/` folder

## Usage

1. Start the backend server
2. Install the browser extension
3. Click the extension icon to see opportunities
4. Click "Place Bets" on any opportunity
5. Extension will auto-fill bet slips
6. Confirm each bet manually

Edit `backend/.env`:
- `ODDS_API_KEY`: Your The Odds API key

- `MIN_EV_PERCENTAGE`: Minimum EV% for +EV bets (default: 5.0)
