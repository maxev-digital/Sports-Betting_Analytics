# ARB Auto Bettor™ - Beta Testing Guide

## Installation Instructions

### Step 1: Download the Extension
1. Download the entire `extension` folder to your computer
2. Remember where you saved it (e.g., `Downloads/ARB_Auto_Bettor/extension`)

### Step 2: Install in Chrome
1. Open Google Chrome
2. Go to `chrome://extensions/`
3. Enable **"Developer mode"** (toggle in top-right corner)
4. Click **"Load unpacked"**
5. Select the `extension` folder you downloaded
6. The ARB Auto Bettor™ icon should appear in your extensions toolbar

### Step 3: Sign Up & Login
1. Go to https://max-ev-sports.com
2. Click "Sign Up" and create an account
3. You'll get a 7-day free trial automatically
4. Login with your credentials

### Step 4: Test the Extension

#### What the Extension Does:
- **Automatically detects arbitrage opportunities** from the Max EV Sports platform
- **Auto-fills bet slips** on supported sportsbooks (95% automated)
- **Sends notifications** when new arb opportunities appear
- **100% ToS compliant** - you still click the final "Place Bet" button

#### Supported Sportsbooks:
- DraftKings
- FanDuel
- BetMGM
- Caesars
- BetRivers
- Bovada
- BetUS
- MyBookie
- BetOnline

#### How to Test:
1. **Open a supported sportsbook** in Chrome (e.g., sportsbook.draftkings.com)
2. **Keep Max EV Sports open** in another tab (https://max-ev-sports.com)
3. **The extension will auto-detect arbs** and fill bet slips
4. **You review and click "Place Bet"** - never fully automatic

### Troubleshooting

**Extension not connecting?**
- Make sure you're logged into https://max-ev-sports.com
- Check the extension icon - it should show "Connected"
- Try refreshing both the sportsbook and Max EV Sports tabs

**Bet slips not auto-filling?**
- Make sure you're on a supported sportsbook
- Check that there are active arbitrage opportunities on Max EV Sports
- Some sportsbooks have CAPTCHAs - solve them first

**Need help?**
Contact support or check the extension console for error messages:
1. Right-click extension icon
2. Select "Inspect popup"
3. Check Console tab for errors
